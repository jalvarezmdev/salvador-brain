# DESCRIPCION
SALVADOR KIDS es el servicio infantil de SALVADOR, diseñado para ofrecer cortes, peinados y cuidado capilar para niños y niñas en un ambiente seguro, cómodo y amigable. Está recomendado para padres que buscan una atención profesional, paciente y delicada para sus hijos.

# SERVICIOS
- Corte de cabello niño / niña: 
    - Descripcion: Corte infantil diseñado para niños y niñas, adaptado a su edad, estilo y tipo de cabello, con una atención cuidadosa, rápida y amigable.
- Peinado sencillo:
    - Descripcion: Peinado práctico y cómodo para el día a día, ideal para mantener una buena presencia y el cabello ordenado de forma natural.
- Corte de flequillos:
    - Descripcion: Retoque o diseño de flequillo para niños y niñas, pensado para mejorar la comodidad, la visibilidad y el estilo del rostro.
- Peinado especializado:
    - Peinado infantil más elaborado, ideal para eventos, cumpleaños, sesiones de fotos o celebraciones especiales.
